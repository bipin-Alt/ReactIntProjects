# To Run this starter kit pnpm+vite+tailwind+shadcnui
``versions:tailwind v4``
``pnpm version:10.10.0``
``tailwind version:4.1.10``

# First Clone this Project
``git clone https://github.com/kashyapsandesh/pnpm-vite_starter_kit.git``

# After Sucessfully clone install pnpm
``pnpm install``

## For installing New shadcn Ui element

pnpm dlx shadcn@latest add card
pnpm dlx shadcn@latest add input
pnpm dlx shadcn@latest add label
pnpm dlx shadcn@latest add slider
pnpm dlx shadcn@latest add checkbox
